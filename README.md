# Interview Prep Hub — Rajesh Dammala

A consolidated collection of interview preparation materials for technical roles.

## 📂 Contents

| File | Description |
|------|-------------|
| `index.html` | **Main Study Plan** — 2-week intensive prep with Python, C#, Terraform, YAML, K8s tabs. Interactive progress tracking. |
| `devops-interview-prep.html` | Microsoft DevOps & SRE interview prep (scripting, pipelines, monitoring) |
| `star-interview-guide.html` | STAR method behavioral interview preparation guide |
| `star-data-analytics-guide.html` | STAR prep for Data Analytics & Support Lead roles |

## 🚀 How to Use

1. Open `index.html` in any browser — no server needed
2. Click through the **14-day schedule** for guided daily study
3. Use **tabbed sections** (Python, C#, Terraform, YAML, K8s) for coding & infra prep
4. **Check off items** as you complete them — progress saves in your browser
5. **Copy code** by hovering over any code block and clicking the copy button

## 💾 Progress Tracking

Progress is saved in your browser's localStorage. It persists between sessions on the same browser. Clear browser data to reset.
